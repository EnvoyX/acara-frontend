hjkl - navigation (normal mode)
xr - making changes while in command (normal mode)

h - move left
j - move down
k - move up
l - move right

x - delete character
r - replace character

iIaA - editing in line
i - insert mode
ctrl + c  or esc - normal mode
I (shift + i) - insert mode at the beginning of the line
A (shift + a) - insert mode at the end of the line
a - append mode after the cursor
